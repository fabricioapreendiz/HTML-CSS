Principais formatações.

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag "B" (não semâtica-sem significado, sendo somente uma forma sem sentido.)

Nesta frase, temos um termo em destaque usando a tag STRONG (semântica).

Itálico/ Ênfase

Nesta frase, temos um termo em itálico usando a tag "I" (não semântica).

Nesta frase, temos um termo em ênfase usando a tag "EM" (semântica). USE ESTA!

Texto marcado.

Podemos criar também um texto marcado usando a tag "MARK".

E no outro parágrafo, temos outro texto marcado no final.

Texto grande e pequeno.

Criando um texto grande e um texto pequeno nesse parágrafo.

Texto deletado.

Podemos marcar um texto como excluído para indicar que ele deve ser lido mas não considerado.

Texto inserido

Podemos marcar um texto como inserido (semântica) para dar uma ênfase e indicar que ele foi adicionado depois.

Outra opção para sublinhar é a tag "U" (não semântica).

Texto sobrescrito.

Usado para escrever coisas do tipo x20+3

Texto subscrito.

Para inserir caractéres do tipo h2o
